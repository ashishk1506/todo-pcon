# todo-pcon

## Endpoints

1.getAllItems -('/') 
  ### METHOD- GET 
   ##### this will give alll the todo items

2.addItem -('/add') 
  ### METHOD- POST 
   ##### this will add new todo item

3.updateById -('/update/:id') 
  ### METHOD- POST 
   ##### this will update the todo list with given id

4.deleteById -('/delete/:id') 
  ### METHOD-POST 
   ##### this will delete the todo item with given id
